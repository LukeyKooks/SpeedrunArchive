About the Game

Boneworks is a VR physics-driven action game. It features a realistic interaction system and experimental combat. The game operates on a unified physics engine, allowing players to manipulate objects, movement, and momentum. This leads to highly creative speedrunning strategies.

Key Speedrunning Features

  • Physics-based movement exploits
  • Prop-flying and object boosting
  • Momentum stacking
  • Out-of-bounds skips
  • Flexible routing via physics freedom

Gameplay Style (Speedrunning)

Boneworks speedrunning focuses on mastering the physics engine. Runners use objects to launch themselves across maps, chain momentum to skip sections, and exploit collision quirks to pass through geometry. The game rewards creativity since almost any object can serve as a movement tool.

Difficulty

Very high skill ceiling. Speedrunning Boneworks requires precise control over physics, quick decision-making, and a deep understanding of object interactions. Small mistakes can derail momentum and cost significant time.
